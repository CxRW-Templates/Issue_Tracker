Welcome to the Issue Tracker REST API. This API provides endpoints for managing issues with create, read, update, and delete operations.

Available Endpoints:

  • POST /api/issues/ - Create a new issue
  • GET /api/issues/ - List all issues (supports ?status= filter)
  • GET /api/issues/{id}/ - Retrieve a specific issue
  • PATCH /api/issues/{id}/ - Update an issue
  • DELETE /api/issues/{id}/ - Delete an issue

Issue Status Options:

  • open
  • in_progress
  • closed
